Descripción del repositorio

Este repositorio contiene DataManagerApp, un proyecto educativo desarrollado para facilitar la comprensión de estructuras de datos lineales. La aplicación aborda una problemática común en estudiantes: la dificultad para visualizar cómo funcionan internamente listas enlazadas, pilas y colas, especialmente durante operaciones clave como inserción, eliminación, recorrido, búsqueda y ordenamiento.
DataManagerApp ofrece una solución interactiva que permite crear, modificar y visualizar estructuras estáticas (arreglos) y dinámicas (listas simples y dobles, pilas y colas). Cada operación ejecutada se muestra de forma clara, acompañada de mensajes informativos que ayudan al usuario a entender qué ocurre “por dentro” en cada estructura.
El proyecto incluye funcionalidades como algoritmos de ordenamiento, búsquedas lineal y binaria, manejo de nodos, y simulación de operaciones típicas como push, pop, enqueue y dequeue. La aplicación está construida en Java, utiliza la consola como interfaz principal, y está diseñada para ser modular, portable y fácil de ampliar.
Además, este repositorio integra buenas prácticas de desarrollo, como control de versiones con GitHub, una organización modular del código y documentación técnica complementaria (requerimientos, historias de usuario y diagramas).
